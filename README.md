# 새 홈페이지

기존 홈페이지의 방향성은 참고하되, 전국학원 영어수학 전문학원 찾기 이름에 맞춰 디자인과 문장을 새롭게 구성한 정적 홈페이지 초안입니다.

## 구성

- `/` 홈
- `/coaching/` 학습코칭
- `/center/` 전국학원
- `/contact/` 상담문의

전국학원 페이지는 `/center/` 슬러그로 추가했고, 추후 지역/센터 목록을 확장할 수 있습니다.

## 배포 전 확인할 것

새 도메인이 확정되면 다음 항목을 추가하거나 수정하는 것이 좋습니다.

- canonical URL
- OG 이미지
- WebSite / Organization / FAQPage JSON-LD
- sitemap.xml
- robots.txt의 Sitemap 주소
